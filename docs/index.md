# Welcome to Ragussy

Ragussy is a universal RAG (Retrieval-Augmented Generation) chatbot that lets you chat with your documentation.

## Features

- **Full Management UI** - Configure everything from the browser
- **Setup Wizard** - Guided configuration for API keys
- **Document Management** - Upload and manage your docs
- **Vector Store Control** - Manage Qdrant collections
- **Chat Interface** - Beautiful chat with source citations

## Quick Links

- [Getting Started](./getting-started) - Installation and setup
- [GitHub Repository](https://github.com/mojomast/ragussy) - Source code

## How It Works

1. **Add Documents** - Upload your markdown documentation
2. **Index** - Ragussy chunks and embeds your docs into vectors
3. **Chat** - Ask questions and get AI-powered answers with sources
